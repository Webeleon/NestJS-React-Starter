## Pre-requisite

- node 14+ LTS

## Configuration

This project use .env file


## Installation

```bash
$ npm install
```

## Building the frontend app

```
npm run build:front:prod
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
